## 起動方法

```bash
# フロントエンド
cd frontend
npm run dev

# バックエンド（別ターミナル）
backend/venv/Scripts/uvicorn backend.main:app --reload
```

- **フロントエンド**: http://localhost:3001
- **バックエンドAPI**: http://127.0.0.1:8000/docs
