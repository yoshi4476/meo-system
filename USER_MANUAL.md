# 🔰 初心者向け：MEO Mastermind AI 利用マニュアル

このマニュアルでは、MEO Mastermind AI（MEOマスターマインドAI）の始め方と使い方をわかりやすく解説します。
このシステムを使うと、Googleマップのお見せの情報をAIが自動で分析したり、クチコミの返信案を考えてくれたりします。

---

## 1. 最初の準備（Googleの設定）

システムがGoogleマップの情報にアクセスできるように、Google側で「鍵」を作る作業が必要です。
少し複雑に見えますが、手順通りに進めれば大丈夫です。

### 手順

1. **Google Cloud Console** (https://console.cloud.google.com/) にアクセスします。
   - Googleアカウントでログインしてください。
2. 画面左上の「プロジェクトの選択」をクリックし、「**新しいプロジェクト**」作成します。
   - 名前は「MEOシステム」など、わかりやすいものでOKです。
3. 検索バーに「**Business Profile API**」と入力して検索し、「有効にする」ボタンを押します。
4. 左メニューの「認証情報」をクリックし、「**認証情報を作成**」→「**OAuth クライアント ID**」を選びます。
5. 「アプリケーションの種類」は「**ウェブ アプリケーション**」を選びます。
6. 「承認済みのリダイレクト URI」という欄に、以下のURLを追加します。
   - `http://localhost:8001/google/callback`
   - ※これは「パソコン内で動かすとき（ローカル環境）」用のアドレスです。
7. 作成完了画面に表示される **「クライアント ID」** と **「クライアント シークレット」** をメモ帳などにコピーしておきます。

---

## 2. システムを動かす（パソコンで起動する）

このシステムは「バックエンド（頭脳）」と「フロントエンド（顔）」の2つのプログラムで動いています。
それぞれ別の画面（黒い画面）で起動する必要があります。

### 準備：設定ファイルの作成

1. ファイルが入っているフォルダを開きます。
2. `backend` フォルダの中に、`.env` という名前の新しいファイルを作ります。
3. そのファイルに、先ほどメモした情報を以下のように書き込んで保存します。

```text
GOOGLE_CLIENT_ID=（コピーしたクライアントIDを貼り付け）
GOOGLE_CLIENT_SECRET=（コピーしたシークレットを貼り付け）
GOOGLE_REDIRECT_URI=http://localhost:8001/google/callback
SECRET_KEY=secret
```

### 起動手順

ここからは「コマンドプロンプト（Windows）」または「ターミナル（Mac）」を使います。

#### ステップ1：バックエンド（頭脳）の起動

1. コマンドプロンプトを開き、システムのフォルダまで移動します。
2. 以下のコマンド（魔法の呪文だと思って入力してください）を順番に実行します。

```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Macの場合は source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8001
```

詳しくはわからなくても、最後に `Application startup complete.` と表示されれば成功です！

#### ステップ2：フロントエンド（顔）の起動

1. **新しい**コマンドプロンプトを開きます（さっきのはそのままにしておいてください）。
2. 同様にフォルダまで移動して、以下のコマンドを実行します。

```bash
cd frontend
npm install
npm run dev
```

最後に `Ready` や `http://localhost:3000` のような文字が出れば成功です！

## 3. 使ってみよう

1. ブラウザ（Chromeなど）を開いて、`http://localhost:3000` にアクセスします。
2. 綺麗なトップページが表示されます。「**Googleでログイン**」ボタンなどはまだ押さずに、まずは体験してみましょう。
3. 画面上のメニューやボタンを操作して、以下の機能を確認できます。

### 🌟 できることリスト

- **ダッシュボード**: お店の調子（検索された回数など）が一目でわかります。
- **投稿を作る**: AIがインスタグラムのような投稿文章を考えてくれます。「夏のキャンペーン」などのキーワードを入れるだけです。
- **クチコミ返信**: お客様からのクチコミに対して、失礼のない丁寧な返信文をAIが提案してくれます。
- **設定**: お店の基本情報を変更したり、Googleアカウントとの連携設定を行ったりします。

## 4. 困ったときは？

- **画面が真っ白になる**:
  - 黒い画面（コマンドプロンプト）にエラーが出ていないか確認してください。
  - プログラムを一度終了（CtrlキーとCキーを同時押し）して、もう一度起動してみてください。

- **Google連携ができない**:
  - 手順1で設定した「リダイレクトURI」が間違っていないか確認してください。
  - `.env` ファイルの内容が合っているか確認してください。

不明点があれば、開発担当までお気軽にご連絡ください。
